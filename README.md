# Ejercicio Bootstrap – Web de Viajes

Proyecto web desarrollado con **Bootstrap 5** como práctica de maquetación, componentes y diseño responsive.

## Descripción
Este ejercicio consiste en la creación de una **web temática de viajes** utilizando Bootstrap, compuesta por una **home pública** y una **página de administración**.

El objetivo es aplicar correctamente el sistema de grid, componentes interactivos y utilidades de Bootstrap para construir una interfaz completa, responsive y estructurada.

---

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- Animate.css

---

## Home pública
La página principal incluye los siguientes elementos:

- Logo
- Selector de idiomas
- Input de búsqueda
- Menú de navegación desplegable y responsive
- Carrusel fotográfico de viajes
- Título / eslogan animado mediante Animate.css
- Dos bloques 50/50 con texto e imagen
- Cuatro bloques informativos responsive
- Bloque con pestañas o acordeones
- Dos bloques responsive:
  - Formulario de login
  - Mapa representado mediante imagen
- Pie de página diferenciado con tres bloques

---

## Página de administración
La página de administración incluye:

- Columna lateral izquierda con logo y menú vertical
- Breadcrumb de navegación
- Tablas de datos
- Botones con ventana modal
- Barra de progreso
- Uso de popovers y tooltips

---

## Diseño responsive
Toda la aplicación es **responsive**, adaptándose a distintos tamaños de pantalla:
- En móvil: los elementos se muestran en una sola columna
- En escritorio: el contenido se distribuye mediante el sistema de grid

El diseño responsive se consigue utilizando clases como `container`, `row`, `col-*`, `d-flex` y utilidades de Bootstrap.

---

## Ejecución del proyecto
1. Descargar o clonar el repositorio
2. Abrir el archivo `index.html` en el navegador

No es necesario instalar dependencias adicionales.
